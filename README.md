# AI-Based-Intrusion-Detection-System
CyberShield is a machine learning-based intrusion detection system that identifies malicious network traffic.
# 🛡️ CyberShield: AI-Based Intrusion Detection System

## 📊 Dataset
UNSW-NB15 dataset from Kaggle

## ⚙️ Features
- Data preprocessing and feature engineering
- Machine learning model comparison
- Cross-validation for reliability
- Feature importance analysis
- Real-time prediction using Streamlit

## 🧠 Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## 📈 Results
- Achieved high accuracy and F1-score
- Random Forest performed best

## 🚀 Deployment
Built interactive web app using Streamlit

## 🛠️ Tech Stack
Python, Scikit-learn, Pandas, Streamlit

## 📌 How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
